2.1. 系统要求

  1. Windows操作系统，例如：Windows 2000、 Windows XP、Windows 2003等。 CDBE假定您的Windows系统安装在C：盘。

  2. JDK或JRE，请到Sun Microsystems, Inc.下载Windows下最新的版本并安装之。

  3. 请到http://sinosmond.com/project/cdbe/start下载CDBE的最新版本。
